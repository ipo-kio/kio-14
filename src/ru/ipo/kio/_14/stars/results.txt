Оценка результатов участников в задаче Звёзды

Для каждого решения участника вычисляется

1. Есть ли пересекающиеся рёбра

   Если какие-то ребра пересекаются, значит, решение считается хуже всех других,
   можно даже не вычислять следующие параметры

2. Количество правильных созвездий.

   В каждом уровне правильность определяется по-разному.
   0 уровень. Правильные созвездия являются деревьями (количество вершин = количество рёбер + 1)
   1 уровень. Правильные созвездия имеют 1 цикл. (количество вершин = количество рёбер)
   2 уровень. Правильные создездия либо деревья, либо имеют один цикл. (т.е. либо как в 0, либо как в 1 уровне)

3. Количество различных созвездий среди правильных. Созвездия считаются одинаковыми с точностью до изоморфизма.

   Это количество буду вычислять Посов, потому что необходим сложный алгоритм. (google: tree isomorphism)

4. Суммарная длина созвездий.

   Считается сумма не всех рёбер, а сумма ребер только в правильных создвездиях.

Сравнение решений участников.

1. Решение с пересекающимися рёбрами хуже всех других. Два решения с пересекающимися ребрами одинаково плохие.
2. Решение с большим количеством различных созвездий считается лучше
3. Если количество различных созвездий в двух решениях одинаковое, то лучшим считается решение, где больше правильных
   созвездий
4. Если количество правильных созвездий тоже совпадает, то лучшим считается решение с меньшей суммой длин созвездий
